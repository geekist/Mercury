# Mercury

基于kotlin语言开发，基于[wanandroid.com](https://www.wanandroid.com/blog/show/2)开放API接口的android app 应用。

## 架构介绍
1、组件化架构

2、基于Kotlin协程的Retrofit + coroutine + LiveData + ViewModel的MVVM架构

3、单activity多fragments

4、依赖注入使用ARouter

5、Databing
